# Ursulinen Combat

Ein Pygame-Spiel entwickelt am Ursulinen Gymnasium Werl.  
Lehrer treten in spannenden Kämpfen gegeneinander an – jeder mit einzigartigen Attacken, Eigenschaften und Fähigkeiten.



## Features
- **Charaktere:** Verschiedene Lehrer mit individuellen Attacken, Sprungkraft und Geschwindigkeit.
- **Level:** Arenen an bekannten Orten der Schule (z. B. Forum, Klassenzimmer).
- **Steuerung:** Frei konfigurierbare Tastenbelegung.
- **Sprachen:** Mehrsprachige Unterstützung für verschiedene Nutzergruppen.
- **Gameplay:** Dynamische Kämpfe mit unterschiedlichen Strategien und Spielstilen.
